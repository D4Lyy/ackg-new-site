# ACKG Website

## Project overview

This repository contains the official website of **ACKG – Association Culturelle Kurde de Genève**.  
The site presents the association’s mission, activities, cultural events, and news, and serves as an information and communication platform for the Kurdish community in Geneva as well as the general public. Its goal is to promote Kurdish culture, social exchange, and community engagement through clear and accessible online content.

## How can I edit this code?

There are several ways to edit and maintain this application.

### Use your preferred IDE (local development)

You can work locally using your own IDE by cloning this repository.

**Requirements**
- Node.js
- npm  
  (Recommended installation via nvm: https://github.com/nvm-sh/nvm#installing-and-updating)

**Steps**

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
